Initializing Cluster Node

Welcome to Mergen.

Stream live, W3C-compliant telemetry into any AI agent environment.

01

Boot the Nexus

Initialize the machine-readable server via CLI. This acts as the local nexus for all telemetry streams.

npx mergen-server@latest setup
02

Bridge to Agent

Register the MCP bridge with Claude Code to enable runtime visibility.

claude mcp add mergen --transport stdio -- node "$(pwd)/server/dist/index.js"

Ensure .cursor/mcp.json provides the correct node path for the server dist.

{
  "mergen": {
    "command": "node",
    "args": ["/path/to/server/dist/index.js"]
  }
}

Inject the mergen configuration into mcp_config.json.

{
  "mcpServers": {
    "mergen": {
      "command": "node",
      "args": ["/path/to/server/dist/index.js"]
    }
  }
}

Activate Agent mode and verify the mergen toolset is initialized via the mcp.json protocol.
