Initializing System

Welcome to Mergen.

Stream live browser telemetry into any AI IDE. Everything stays on 127.0.0.1.

01

Boot the Server

Initialize the machine-readable stack via CLI:

npx mergen-server@latest setup
02

Link MCP IDE

Register the MCP server with Claude Code:

claude mcp add mergen --transport stdio -- node "$(pwd)/server/dist/index.js"

Ensure .cursor/mcp.json contains the mergen entry:

{
  "mergen": {
    "command": "node",
    "args": ["/path/to/server/dist/index.js"]
  }
}

Update ~/.codeium/windsurf/mcp_config.json:

{
  "mcpServers": {
    "mergen": {
      "command": "node",
      "args": ["/path/to/server/dist/index.js"]
    }
  }
}

Enable Agent mode in Copilot Chat and ensure the mergen tool is active via the mcp.json configuration.

03

Instrument Runtime

Mergen records all console.*, fetch, and XHR signals automatically on any open tab.

04

Consult Agent

Ask your AI about the runtime state. It now has eyes on your execution flow.
