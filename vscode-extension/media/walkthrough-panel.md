# Step 3 — Open the Mergen panel

Click the Mergen icon in the activity bar. With the server running and the
browser extension loaded, **refresh any tab in your browser** — the panel
should light up within ~2 seconds:

- **Buffer** — live counts of console / network events captured
- **Context Pack** — the current top hypothesis (HIGH / MEDIUM / LOW
  confidence) plus the suggested fix
- **History** — your last 20 diagnoses, labeled by trigger reason
  (`pageload`, `hmr`, `net_burst`, `error`, …)

If you don't see anything, run **Mergen: Send Feedback** from the command
palette — the first 50 installs are how we shape the onboarding flow.
